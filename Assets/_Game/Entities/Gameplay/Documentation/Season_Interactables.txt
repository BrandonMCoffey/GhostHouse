# Seasonal Interactables
Place any season-specific interactables here. Please work inside of the appropriate prefabs or apply changes.
Universal Interactables can be found under this object. If making changes to the Universal one, override changes to the Seasonal Prefab, open it, then override changes to the Universal Prefab as well.
### Notes
This object will mainly be worked on by Level Design. Animation or Art may add stuff to prefabs inside of the object, but should apply any changes made.
